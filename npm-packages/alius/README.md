# @aliustech/alius

Alius CLI - AI-powered assistant with tool calling capabilities.

## Installation

```bash
npm install -g @aliustech/alius
```

## Usage

```bash
alius --help
alius repl          # Start interactive REPL mode
alius run "prompt"  # Run a single prompt
alius config        # Configuration management
alius version       # Show version info
```

## Features

- 🤖 AI-powered CLI assistant
- 🔧 Built-in tools (file operations, shell, git, HTTP, etc.)
- 💬 Interactive REPL mode
- 📝 Session management
- 🔐 Permission system for tool execution

## Supported Platforms

- macOS (x64, arm64)
- Linux (x64)
- Windows (x64)

## Requirements

- Node.js >= 14
