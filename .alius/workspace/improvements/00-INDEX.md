## 📊 竞品对比总结

### free-code
- **定位**: 简化的 AI 编码助手
- **特点**: 轻量级、快速启动
- **借鉴**: 简洁的命令设计、快速响应

### claude-code
- **定位**: Anthropic 官方 CLI 的社区重构
- **特点**: 
  - Bun + TypeScript 技术栈
  - React Ink TUI 框架
  - 完整的 MCP 支持
  - OpenTelemetry 可观测性
  - Agent SDK 集成
- **借鉴**: 
  - MCP 协议实现模式
  - OpenTelemetry 集成方案
  - React 风格的 TUI 组件设计
  - 多云模型提供商抽象

### claw-code
- **定位**: Rust agent harness
- **特点**:
  - 纯 Rust 实现
  - Agent-managed 理念
  - 关联 harness 生态
- **借鉴**:
  - Rust 项目组织模式
  - Agent harness 设计模式
  - 自动化维护流程

### codex (OpenAI)
- **定位**: OpenAI 官方 Codex CLI
- **特点**:
  - Node.js + Rust 混合架构
  - pnpm monorepo
  - 完整的桌面应用模式
  - 多平台分发
- **借鉴**:
  - Monorepo 组织结构
  - 跨平台构建流程
  - 安装脚本设计
  - SDK 多语言绑定
